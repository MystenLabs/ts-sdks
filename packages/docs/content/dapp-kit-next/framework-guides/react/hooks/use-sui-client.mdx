---
title: useSuiClient
---

The `useSuiClient` hook provides access to the SuiClient instance for the current network.

<Callout type="warn">
    All dApp Kit hooks must be used within components that are descendants of [DAppKitProvider](./dapp-kit-provider). Using them outside will result in an error.
</Callout>

## Usage

```tsx
import { useSuiClient } from '@mysten/dapp-kit-react';

function MyComponent() {
  const client = useSuiClient();
  
  const handleQuery = async () => {
    const balance = await client.getBalance({
      owner: '0x...',
    });
    console.log(balance);
  };
  
  return <button onClick={handleQuery}>Check Balance</button>;
}
```

## Return Value

```tsx
SuiClient
```
