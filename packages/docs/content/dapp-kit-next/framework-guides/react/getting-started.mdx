title: Getting Started in React
---

This guide shows you how to integrate dApp Kit Next into React applications.

## Installation

```sh npm2yarn
npm i @mysten/dapp-kit-react @mysten/dapp-kit-core @mysten/sui
```

Create a dApp Kit instance and wrap your app with the provider:

```tsx
// dapp-kit.ts
import { createDAppKit, DAppKitProvider } from '@mysten/dapp-kit-react';
import { SuiClient, getFullnodeUrl } from '@mysten/sui/client';

export const dAppKit = createDAppKit({
	networks: ['mainnet', 'testnet'],
	defaultNetwork: 'mainnet',
	createClient: (network) => {
		return new SuiClient({ url: getFullnodeUrl(network) });
	},
});

declare module '@mysten/dapp-kit-react' {
	interface Register {
		dAppKit: typeof dAppKit;
	}
}
```

```tsx
// App.tsx
import { dAppKit } from './dapp-kit';

export default function App() {
	return (
		<DAppKitProvider dAppKit={dAppKit}>
			<YourApp />
		</DAppKitProvider>
	);
}
```

For more details on the provider, see the [DAppKitProvider documentation](./dapp-kit-provider).

## Using Components

The React package provides [React components](./components/connect-button) that wrap the Web
Components:

```tsx
import { ConnectButton } from '@mysten/dapp-kit-react';
```

## Using React Hooks

Access dApp Kit functionality through hooks:

```tsx
import { useCurrentAccount, useCurrentWallet, useCurrentNetwork } from '@mysten/dapp-kit-react';

export function WalletStatus() {
	const currentAccount = useCurrentAccount();
	const currentWallet = useCurrentWallet();
	const currentNetwork = useCurrentNetwork();

	if (!currentAccount) {
		return <div>Not connected</div>;
	}

	return (
		<div>
			<p>Wallet: {currentWallet?.name}</p>
			<p>Address: {currentAccount.address}</p>
			<p>Network: {currentNetwork}</p>
		</div>
	);
}
```
