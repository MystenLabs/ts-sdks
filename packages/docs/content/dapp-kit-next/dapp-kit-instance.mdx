title: DApp Kit Instance

The `createDAppKit` function is the foundation of your Sui dApp. It creates an instance that manages
wallet connections, network configuration, and provides access to the Sui client.

## Creating a dApp Kit Instance

The core of dApp Kit Next is the `createDAppKit` function, which creates an instance that manages
all dApp functionality:

```typescript
import { createDAppKit } from '@mysten/dapp-kit-core';
import { SuiClient, getFullnodeUrl } from '@mysten/sui/client';

export const dAppKit = createDAppKit({
	networks: ['mainnet', 'testnet'],
	defaultNetwork: 'mainnet',
	createClient: (network) => new SuiClient({ url: getFullnodeUrl(network) }),
});
```

This instance provides:

- Wallet connection management
- Network switching capabilities
- Client access for blockchain interactions
- State stores for reactive updates

## Parameters

- **networks** - Array of network names your dApp supports (e.g., `['mainnet', 'testnet']`)
- **createClient** - Function that creates a SuiClient for each network
- **defaultNetwork** (optional) - Initial network to use (default: first network in the array)
- **autoConnect** (optional) - Automatically reconnect to previous wallet (default: `true`)
- **enableBurnerWallet** (optional) - Enable development-only burner wallet (default: `false`)
- **storage** (optional) - Storage for connection persistence (default: `localStorage`, set to
  `null` to disable)
- **storageKey** (optional) - Key prefix for storage (default:
  `'mysten-dapp-kit:selected-wallet-and-address'`)
- **walletInitializers** (optional) - Array of custom wallet integrations
- **slushWalletConfig** (optional) - Configuration for Slush wallet (set to `null` to disable)

## Return Value

```ts
{
    // Actions
    connectWallet: (options: ConnectWalletOptions) => Promise<void>;
    disconnectWallet: () => Promise<void>;
    switchAccount: (account: UiWalletAccount) => Promise<void>;
    switchNetwork: (network: string) => void;
    signTransaction: (options: SignTransactionOptions) => Promise<SignedTransaction>;
    signAndExecuteTransaction: (options: SignAndExecuteOptions) => Promise<SuiTransactionResponse>;
    signPersonalMessage: (options: SignMessageOptions) => Promise<SignedMessage>;

    // Properties
    networks: string[];
    stores: DAppKitStores;
    getClient: (network?: string) => SuiClient;
}
```

Read more about [actions](./actions/connect-wallet) and [stores](./state).

## Basic Usage

```ts
// dapp-kit.ts
import { createDAppKit } from '@mysten/dapp-kit-core';
import { SuiClient } from '@mysten/sui/client';

export const dAppKit = createDAppKit({
	networks: ['mainnet', 'testnet'],
	defaultNetwork: 'mainnet',
	createClient: (network) => {
		const connections = {
			mainnet: { url: 'https://mainnet.sui.io:443' },
			testnet: { url: 'https://testnet.sui.io:443' },
		};
		return new SuiClient(connections[network]);
	},
});
```

```ts
// connect-wallet-and-log.ts
import type { DAppKit, UiWallet, UiWalletAccount } from '@mysten/dapp-kit-core';

export async function connectWalletAndLog({
	dAppKit,
	wallet,
	account,
}: {
	dAppKit: DAppKit;
	wallet: UiWallet;
	account: UiWalletAccount;
}) {
	const result = await dAppKit.connectWallet({
		wallet,
		account,
	});

	console.log('Connected accounts:', result.accounts);
}
```

## Development Configuration

For development environments, enable the burner wallet:

```typescript
import { createDAppKit } from '@mysten/dapp-kit-core';
import { SuiClient } from '@mysten/sui/client';

export const dAppKit = createDAppKit({
	networks: ['localnet'],
	defaultNetwork: 'localnet',
	createClient: () => new SuiClient({ url: 'http://localhost:9000' }),
	enableBurnerWallet: true, // Dev-only feature
});
```

## Auto-Connect Behavior

By default, dApp Kit automatically reconnects to the previously connected wallet.

### Disabling Auto-Connect

```typescript
import { createDAppKit } from '@mysten/dapp-kit-core';
import { SuiClient, getFullnodeUrl } from '@mysten/sui/client';

export const dAppKit = createDAppKit({
	networks: ['mainnet'],
	createClient: (network) => new SuiClient({ url: getFullnodeUrl(network) }),
	autoConnect: false,
});
```

## Custom Wallet Integration

Add support for custom wallets using wallet initializers:

```typescript
import { createDAppKit } from '@mysten/dapp-kit-core';
import { SuiClient, getFullnodeUrl } from '@mysten/sui/client';
import { getWallets } from '@mysten/wallet-standard';
import { CustomWallet } from './custom-wallet.js'; // import your custom wallet class

export const dAppKit = createDAppKit({
	networks: ['mainnet'],
	createClient: (network) => new SuiClient({ url: getFullnodeUrl(network) }),
	walletInitializers: [
		{
			id: 'Custom Wallets',
			initialize() {
				const wallets = [new CustomWallet()];
				const walletsApi = getWallets();
				return { unregister: walletsApi.register(...wallets) };
			},
		},
	],
});
```
